가계부 개인정보처리방침

시행일: 2026년 6월 19일

가계부는 사용자의 생활 장부를 기기 중심으로 관리하도록 설계된 앱입니다. 본 방침은 가계부 앱에서 처리될 수 있는 정보와 사용 방식을 설명합니다.

처리하는 정보

  • 사용자가 직접 입력한 거래 금액, 날짜, 상점 또는 상대방, 카테고리, 계정, 메모, 태그, 첨부 정보
  • 검토 상태, 중복 의심 처리, 반복 지출 변화 처리, 마감 상태
  • 앱 설정, 언어, 테마, 위젯 개인정보 표시 설정
  • 영수증 또는 음성 입력 기능을 사용하는 경우 사용자가 제공한 이미지, 음성 또는 초안 내용

정보 사용 목적

  • 가계부 기록, 수정, 분류, 검토 기능 제공
  • 중복 기록, 카테고리 누락, 반복 지출 변화 등 사용자 확인 항목 표시
  • 계획, 마감, 위젯, 언어와 테마 설정 유지
  • 사용자가 선택한 입력 보조 기능 제공

저장 및 전송

핵심 데이터는 기본적으로 사용자의 기기에 저장됩니다. 가계부는 계정 생성이나 개발자 서버 저장을 전제로 하지 않으며, 사용자의 거래 기록을 개발자 서버로 전송하지 않습니다.

권한

영수증 촬영, 이미지 첨부, 음성 입력, 알림, 위젯 기능을 사용하는 경우 기기 권한이 필요할 수 있습니다. 권한은 해당 기능 제공을 위해서만 사용됩니다.

광고

무료 버전에서 제한적인 광고가 표시될 수 있습니다. 광고는 핵심 기록, 검토, 마감 결정 화면을 방해하지 않는 위치에만 표시하는 것을 원칙으로 합니다. 거래 금액, 상점명, 계정명, 메모, 영수증 내용은 광고 타겟팅 문맥으로 전달하지 않습니다.

분석 및 제3자 제공

현재 이 공개 방침은 로컬 우선 앱 동작을 기준으로 작성되었습니다. 향후 분석, 동기화, 백업, 광고 SDK 등 제3자 서비스가 추가될 경우 앱 스토어 고지와 본 방침을 업데이트합니다.

보관 및 삭제

앱 내 데이터는 사용자가 앱에서 삭제하거나 앱을 삭제할 때 기기에서 제거될 수 있습니다. 운영체제의 백업 설정에 따라 기기 백업에 포함될 수 있습니다.

아동의 개인정보

가계부는 아동을 대상으로 하지 않습니다. 보호자가 아동의 개인정보가 입력되었다고 판단하는 경우 아래 문의처로 연락해 주세요.
